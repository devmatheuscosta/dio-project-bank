## Getting Started

### Prerequisites

**TypeScript**: `version 5.5.4`

### Installation

Build the project from source:

1. Clone the dio-project-bank.git repository:

```sh
❯ git clone https://github.com/devmatheuscosta/dio-project-bank.git
```

2. Navigate to the project directory:

```sh
❯ cd dio-project-bank.git
```

3. Install the required dependencies:

```sh
❯ npm install
```

### Usage

To run the project, execute the following command:

```sh
❯ npm run start:dev
```
